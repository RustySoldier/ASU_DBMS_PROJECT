
## Clone a repository:
git clone https://github.com/RustySoldier/ASU_DBMS_PROJECT.git

## To add something to the repository

###Add the files you want to upload in the folder with .git file

### Stay in the folder that has the .git file

1. git status ---> gives u status of all the files that are there updated/or up to date

2. git add . ---> adds all the file to start the upload process

3. git commit -m 'Give comment about whats the commit about' 

4. git push ---> merge the files to github


## To get new files from the github server back to ur local machine

### stay in the folder with .git file

1. git pull
